# 🎓 Student Placement Prediction

A Machine Learning web application that predicts whether a student is likely to get placed based on academic and skill-based attributes.

## 🚀 Features

- Student placement prediction
- Random Forest Classifier
- Interactive Gradio web interface
- Probability score
- Easy deployment on Hugging Face Spaces

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Gradio
- Joblib

## ▶️ Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```

## 📊 Machine Learning

- Algorithm: Random Forest Classifier
- Data Preprocessing:
  - Missing value handling
  - One-Hot Encoding
  - StandardScaler
- Evaluation:
  - Accuracy
  - Classification Report
  - Confusion Matrix
